## 📚 BỘ LUẬT CƠ BẢN (Ưu tiên cao nhất)

### 1. **Bộ luật Dân sự 2015** ⭐⭐⭐⭐⭐
- **Tên file:** `Bo_luat_Dan_su_2015.pdf`
- **Lý do:** Nền tảng của mọi hợp đồng dân sự
- **Nội dung quan trọng:**
  - Phần VI: Nghĩa vụ và hợp đồng (Điều 385-407)
  - Hợp đồng mua bán (Điều 453-490)
  - Hợp đồng cho thuê tài sản (Điều 574-600)
  - Hợp đồng dịch vụ (Điều 652-676)
  - Hợp đồng vay (Điều 466-483)
  - Điều kiện hợp lệ của hợp đồng (Điều 385-392)

### 2. **Luật Thương mại 2005** ⭐⭐⭐⭐⭐
- **Tên file:** `Luat_Thuong_mai_2005.pdf`
- **Lý do:** Quy định về hợp đồng thương mại
- **Nội dung quan trọng:**
  - Chương II: Hợp đồng mua bán hàng hóa
  - Chương III: Đại diện thương mại
  - Chương IV: Khuyến mại
  - Chương V: Hoạt động thương mại điện tử

### 3. **Bộ luật Lao động 2019** ⭐⭐⭐⭐⭐
- **Tên file:** `Bo_luat_Lao_dong_2019.pdf`
- **Lý do:** Phân tích hợp đồng lao động
- **Nội dung quan trọng:**
  - Chương III: Hợp đồng lao động (Điều 13-48)
  - Thời giờ làm việc, nghỉ ngơi (Điều 98-118)
  - Tiền lương (Điều 90-97)
  - Kỷ luật lao động (Điều 119-131)
  - Chấm dứt hợp đồng (Điều 34-48)

## 🚀 Quy trình Train AI:

### Bước 1: Download bộ luật
```bash
# Tải 5 bộ luật CƠ BẢN trước (ưu tiên cao nhất)
1. Bộ luật Dân sự 2015
2. Luật Thương mại 2005
3. Bộ luật Lao động 2019
4. Luật Đất đai 2013
5. Luật Nhà ở 2014
```

### Bước 2: Chuyển đổi format (nếu cần)
```bash
# Nếu file là DOC/DOCX, convert sang PDF
# Hoặc extract text sang TXT
```

### Bước 3: Chạy ingest script
```bash
cd backend
python ingest.py
```

### Bước 4: Verify vector store
```bash
python setup_knowledge.py
```

### Bước 5: Test AI agent
```bash
python test_agent.py
```

## 💡 Lưu ý quan trọng:

1. **Ưu tiên 5 bộ luật cơ bản** - Đủ để phân tích 80% hợp đồng phổ biến
2. **Format PDF tốt nhất** - Dễ parse và extract text
3. **Tổng dung lượng:** Khoảng 200-500 MB cho tất cả
4. **Thời gian train:** 30-60 phút tùy số lượng bộ luật
5. **Vector store size:** ~1-2 GB sau khi train xong
